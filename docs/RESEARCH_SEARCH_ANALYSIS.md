## 2. Предложения по улучшению релевантности

### 2.1 Расширить негативный словарь (NON_PSYCHOLOGY_TERMS)

Добавить термины из обнаруженных нерелевантных областей:

```javascript
// Медицина (не психология)
'spine', 'vertebral', 'surgery', 'surgical', 'позвоночник', 'хирургический',
'cardiac', 'myocardial', 'echocardiography', 'ischemia', 'миокард', 'эхокардиография',

// IT/AI (не психология)
'artificial intelligence', 'machine learning', 'cloud service', 'algorithm',
'искусственный интеллект', 'машинное обучение', 'облачный сервис', 'алгоритм',
'neural network', 'deep learning', 'нейронная сеть', 'глубокое обучение',

// Лингвистика/педагогика (не психология)
'language learning', 'pronunciation', 'grammar', 'english teaching',
'изучение языка', 'произношение', 'грамматика', 'преподавание английского',

// История/политика (не когнитивная/клиническая психология)
'historical memory', 'political identity', 'migration policy',
'историческая память', 'политическая идентичность',

// Орнитология/зоология
'bird migration', 'ornithology', 'миграция птиц', 'орнитология',
```

### 2.2 Расширить негативный словарь журналов (NON_PSYCHOLOGY_VENUES)

```javascript
// Медицинские журналы (не психиатрия)
'spine surgery', 'cardiology', 'хирургия позвоночника', 'кардиология',

// Технические журналы
'information technologies', 'program systems', 'информационные технологии',

// Исторические/политические
'власть', 'revue des études', 'славянские исследования',
```

### 2.3 Использовать OpenAlex Concepts API

OpenAlex присваивает каждой работе **concepts** (концепты) с уровнем уверенности. Можно фильтровать по:

```
filter=concepts.id:C15744967  // Psychology
filter=concepts.id:C169760540 // Developmental psychology
filter=concepts.id:C161191863 // Clinical psychology
```

**Преимущество:** Встроенная классификация OpenAlex на основе ML.

### 2.4 Добавить контекстную дисамбигуацию

Для амбивалентных терминов автоматически добавлять контекст:

| Запрос | Дисамбигуация |
|--------|---------------|
| память | память когнитивная |
| обучение | обучение психология |
| стресс | стресс психологический |
| фиксация | фиксация психоанализ |
| сопротивление | сопротивление терапия |

### 2.5 Двухэтапная фильтрация

1. **Этап 1:** Получить больше результатов (100+)
2. **Этап 2:** Отранжировать по психологическому скору
3. **Этап 3:** Вернуть топ-N с самым высоким скором

## 3. Улучшения UX (лучшие практики)

### 3.1 Фасетная фильтрация (как на PubMed, Google Scholar)

- **По типу исследования:** эмпирическое, обзор, мета-анализ
- **По субдисциплине:** клиническая, когнитивная, развития, социальная
- **По методологии:** эксперимент, опрос, лонгитюд, кейс-стади
- **По году:** слайдер (уже есть!)
- **По языку:** чекбоксы (уже есть!)

### 3.2 Подсказки при вводе (autocomplete)

Показывать подсказки из:
- Популярных психологических терминов
- Предыдущих запросов пользователя
- MeSH/APA терминов

### 3.3 "Вы имели в виду?" (disambiguation)

Для амбивалентных терминов показывать:

```
Результаты по запросу "память"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Возможно, вы искали:
• память когнитивная (психология)
• историческая память (история)
• компьютерная память (IT)
[Показать все]
```

### 3.4 Highlighting ключевых слов

Подсвечивать найденные термины в заголовках и аннотациях (как на Google).

### 3.5 Сортировка

- По релевантности (по умолчанию)
- По дате (новые первыми)
- По цитированиям
- По альтметрикам

### 3.6 Экспорт результатов

- BibTeX
- RIS (для Mendeley, Zotero)
- CSV

### 3.7 Сохранение поисков

- История поиска
- Закладки статей
- Оповещения о новых статьях по запросу

### 3.8 Расширенный поиск

Форма с полями:
- Заголовок содержит
- Автор
- Журнал
- DOI
- Год от-до
- Только Open Access (уже есть)

### 3.9 Связанные статьи

Для каждого результата показывать "Похожие статьи" (через OpenAlex related_works).

### 3.10 Превью абстракта

При наведении/клике показывать полный абстракт без перехода на внешний сайт.
