# Продуктовые идеи и геймификация

> Бэклог идей для улучшения UX, вовлечения студентов и геймификации.
> Статусы: `[ ]` — идея, `[~]` — в работе, `[x]` — реализовано

---

## Страница профиля

### Для незалогиненных (гостей)

- [ ] **Превью курсов с прогресс-баром** — показать структуру каждого курса (сколько занятий, какие темы) с визуальным "0% пройдено", чтобы мотивировать начать

- [ ] **Бесплатный демо-тест** — дать пройти один тест без регистрации, а результат показать после входа ("Ваш результат сохранён, войдите чтобы увидеть")

- [ ] **Отзывы/цитаты студентов** — социальное доказательство: "Мне курс помог понять своего ребёнка" с аватарками

- [ ] **Счётчик сообщества** — "Уже 247 студентов изучают психологию развития" (живая статистика)

- [ ] **Карточки "Что вы получите"** — визуально показать: видео-лекции, тесты для самопроверки, личные заметки, таймлайн жизни

### Для залогиненных студентов

- [ ] **Прогресс по курсу** — визуальная шкала "Пройдено 4 из 15 занятий", какие темы изучены, какие впереди

- [ ] **Streak / серия дней** — "Вы учитесь 5 дней подряд!" — геймификация для мотивации

- [ ] **Рекомендация "Продолжить"** — карточка с последним незавершённым занятием и кнопкой "Продолжить с того места"

- [ ] **Статистика тестов** — средний балл, сильные/слабые темы, график прогресса по времени

- [ ] **Достижения/бейджи** — "Прошёл первый тест", "Изучил младенчество", "Создал 10 заметок" — коллекционирование

## Геймификация

### Прогресс и уровни

- [ ] **Уровни эксперта по возрастам** — "Эксперт по младенчеству", "Знаток подросткового возраста" — отдельный уровень за каждый пройденный период с иконкой-наградой

- [ ] **Общий уровень психолога** — Новичок → Студент → Бакалавр → Магистр → Профессор — растёт по мере прохождения всех курсов

- [ ] **Опыт (XP) за действия** — +10 XP за просмотр лекции, +20 XP за тест, +5 XP за заметку, +50 XP за 100% на тесте

### Соревновательность

- [ ] **Таблица лидеров недели** — топ-10 студентов по набранным очкам за неделю (опционально, можно скрыть себя)

- [ ] **Челленджи** — "Пройди 3 теста за неделю", "Изучи 2 новых периода" — с наградой за выполнение

- [ ] **Дуэли на знания** — пригласить друга и соревноваться кто лучше ответит на 10 случайных вопросов

### Коллекционирование

- [ ] **Карточки авторов** — собирать карточки Выготского, Пиаже, Эриксона после изучения их теорий (как коллекция)

- [ ] **Созвездие знаний** — визуальная карта всех тем как звёзды, изученные "загораются" — видно прогресс

- [ ] **Секретные достижения** — скрытые награды: "Сова" (учился после полуночи), "Марафонец" (5 лекций за день), "Перфекционист" (100% на 5 тестах подряд)

### Персонализация

- [ ] **Кастомизация профиля за достижения** — разблокировать рамки для аватара, темы оформления, титулы ("Знаток эмоций", "Исследователь памяти") которые можно показывать

### Специфичные для психологии

- [ ] **"Пройди свой путь"** — заполнить таймлайн своей жизни и получить бейдж за каждый описанный период

- [ ] **Рефлексия недели** — еженедельный вопрос "Что нового вы узнали о себе?" с наградой за ответ
