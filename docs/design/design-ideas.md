# Design ideas

## Короткое позиционирование UX
Текущая визуальная система лучше всего работает для спокойного, вдумчивого обучения и саморефлексии. Восприятие ближе к «бережной образовательной среде», а не к агрессивному EdTech‑маркетингу.

## Сильные стороны
- Мягкий визуальный тон: пастели, округлости, светлый фон.
- Ясная структура: карточки, табы, понятные CTA.
- Таймлайн визуально выделяется и запоминается.

## Риски и ограничения
- Несогласованность стиля между лендингом и продуктовой частью.
- Слишком много локальных цветов и ручных хексов.
- Эмодзи используются как системные иконки, что снижает «профессиональность».
- Контраст вторичных текстов может быть недостаточным на малых размерах.

## Идеи улучшений (приоритетные)
1) Унификация палитры
- Связать цвета лендинга с глобальными токенами.
- Ввести семантические токены: `success`, `warning`, `danger`, `info` и их light‑варианты.

2) Единая типографика
- Оставить Manrope как основной, Georgia — только как акцентный шрифт.
- Зафиксировать типографическую шкалу для заголовков и текста.

3) Иконки вместо эмодзи в интерфейсе
- Эмодзи оставить для учебного контента.
- Системные элементы (ошибки, меню, навигация) — SVG‑иконки.

4) Состояния компонентов
- Стандартизировать `hover`, `active`, `focus-visible`, `disabled`.
- Добавить единые классы в `src/index.css` для интерактива.

5) Доступность
- Проверить контраст у вторичных текстов и ссылок.
- Ввести размерные лимиты для текста и подписей.

## Идеи для развития визуального бренда
- Добавить «бренд‑паттерн» (легкий фон‑шум, геометрия или волна).
- Использовать один фирменный градиент, повторяющийся в ключевых CTA.
- Держать один визуальный «ритм»: одинаковые углы, отступы, высоты кнопок.

## Идеи для таймлайна
- Сделать иконки событий векторными (SVG) для резкости на любых масштабах.
- Добавить слой «легенды» с цветами сфер.
- Визуально показать «ветки» через легкий фон или прозрачный бейдж.

## Идеи для контентных страниц
- Унифицировать блоки «Видео / Литература / Понятия» через один шаблон.
- Добавить «подзаголовочный» стиль, чтобы улучшить сканирование.
- Выравнивать основные CTA по одной вертикали (особенно на Period pages).

## Идеи для тестов
- Поддержать темы тестов с контролируемым набором градиентов, чтобы не было «случайных» цветов.
- Добавить визуальную шкалу прогресса в стиле общего бренда.

## Быстрые wins
- Вынести хексы из `HomePage.tsx` в токены.
- Удалить эмодзи из навигационных кнопок и заменить на SVG.
- Добавить `focus-visible` для всех кнопок и ссылок.
