# Таблица по расстройствам (Disorder Table)

> **Дата:** 2026-04-16
> **Статус:** Актуальный guide

Матричный инструмент для студентов клинической психологии: записи наблюдений на пересечении психопатологических функций (строки) и типов расстройств (столбцы).

---

## Обзор

- **Маршрут:** `/disorder-table`
- **Доступ:** RequireAuth + `courseAccess.clinical === true`
- **Firestore:** `disorderTables/{userId}_{courseId}/entries/{entryId}`
- **Lazy loading:** Да (через `src/pages/lazy.ts`)

## Функциональность

### Основная таблица
- Фиксированный заголовок со строками и столбцами
- В ячейке показан preview первой записи (до 120 символов) + бейдж "+N ещё"
- Цвет ячейки зависит от track записей (диагональный градиент при mixed)

### Фильтры
- Draft-фильтры (не применяются до нажатия "Применить фильтр")
- Фильтрация по строкам и/или столбцам — логика OR
- Кнопка "Сбросить" — очистка фильтров и поиска

### Поиск
- Case-insensitive substring match по тексту записей
- Панель результатов с preview и навигацией к ячейке
- Подсветка найденного фрагмента

### Режим выбора ячеек (только desktop)
- Множественный выбор ячеек для batch-ввода
- Визуальная индикация: amber ring + checkmark
- "Внести текст в выбранные (N)" — bulk-создание записей

### Модальные окна
1. **Cell Viewer** — просмотр всех записей в ячейке (на mobile read-only)
2. **Entry Editor** — создание/редактирование одной записи (текст 3–4000 символов + track)
3. **Bulk Entry** — ввод текста в несколько ячеек одновременно

### Mobile
- Таблица скроллится горизонтально
- Просмотр записей доступен, редактирование — нет
- Сообщение: "Редактирование доступно на компьютере"

## Валидация

- Ровно 1 строка на запись (`rowIds.length === 1`)
- 1+ столбцов (`columnIds.length >= 1`)
- Текст: минимум 3, максимум 4000 символов (после trim)
- Track: `patopsychology`, `psychiatry` или `null`
